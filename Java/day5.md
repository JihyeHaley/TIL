### for each

for(Java 5) : for each 문

- 배열 또는 컬렉션 객체의 데이터들을 꺼내서 반복처리하려는 경우 사용하는 반복문이다.
- 앞에서부터 차례대로 하나하나 꺼내서 처리하려는 경우!

```java
for(변수선언 : 배열 또는 콜렉션 객체)]
     반복문장; 

for( int data : score)
     sum += data;
```
